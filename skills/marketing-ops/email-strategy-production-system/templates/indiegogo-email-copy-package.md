# Indiegogo Campaign Email Copy Package — Template

## 5-Track Structure

For a time-bound crowdfunding campaign (Indiegogo/Kickstarter), structure the email copies as 5 tracks:

| Track | Emails | Timing | Goal |
|-------|--------|--------|------|
| **A: Welcome/Nurture** | 6 | Subscribe to L-3 days | Educate, build trust, install value math |
| **B: Launch Week** | 4-5 | L-1 to L+3 | Convert warmed list on Day 1-2 |
| **C: Mid-Campaign** | 5 | L+5 to L+20 | Sustain momentum, inject new proof |
| **D: Last 48 Hours** | 3 | T-48h to close | Final urgency push |
| **E: Post-Launch** | 3 | L+3 to L+30+ | Backer welcome, referral, production |

## Per-Email Output Structure

Each email in the master file must include ALL of these sections:

```markdown
### EMAIL [ID] — [Name]

#### Quick Reference
- **Timing:** [relative to subscribe or launch]
- **Segment:** [who receives this]
- **Objective:** [what this email achieves]
- **Proof type:** [spec / story / social proof / authority / math]
- **CTA type:** [learn / confirm / buy / share]

#### Subject Line
[1-2 options, max 50 characters, finished copy]

#### Preview Text
[1 option, max 90 characters]

#### Body Copy
[Full email body. 150-400 words. Connected sentences.
One CTA link/button at the end. Paste-ready. No placeholders.
No "see above."]

#### Visual Brief
- **Layout type:** [plain text / designed hero + text / designed box]
- **Hero image concept:** [description of what the hero image shows]
- **Color palette:** [hex codes for primary, secondary, accent]
- **Typography:** [heading font, body font, sizes]
- **Mobile considerations:** [how this email should look on phone]

#### Designer Instructions
- **Image dimensions:** [width x height for hero image]
- **Button style:** [full-width rounded / inline pill / outline]
- **Spacing notes:** [padding, margin, layout rules]
- **Font stack:** [specific fonts to use]

#### AI Image Generation Brief
- **Prompt:** [detailed text-to-image prompt]
- **Style:** [photorealistic / editorial / product shot / lifestyle]
- **Ref:** [lighting, mood, framing]
- **Aspect ratio:** [e.g. 1200x800]
```

## Creative Brief (Top of Master File)

Include at the top of the complete copy package:

1. **Brand voice** in one paragraph
2. **Words to use** (lexicon)
3. **Words to avoid** (forbidden list)
4. **Color palette** with hex codes (primary, secondary, accent, text, background)
5. **Photography style guide** (lighting, POV, environments, color grade)
6. **Designer do's and don'ts**
7. **Font stack** (headings, body, numbers)
8. **Global email template rules** (max width, background, footer, CTA rules)

## Writing Quality Rules (Mandatory, Enforced in All Copy)

- NO em dashes. Use periods, commas, or sentence breaks.
- NO staccato fragments. Connect sentences with "because," "so," "and," "which."
- NO Tier 1 AI vocabulary (delve, tapestry, beacon, cutting-edge, seamless, game-changer, leverage, robust).
- NO exclamation marks. Zero.
- NO rhetorical questions.
- NO "it's not X, it's Y" pattern. State everything as direct positive claim.
- NO "your wife." Use "your partner."
- Vary sentence and paragraph lengths. Read-aloud test must pass.

## Track-Specific Notes

### Track A (Welcome/Nurture)
- W1: Immediate on subscribe. Confirm decision, set expectations, seed the promise.
- W2-W3: Build emotional connection and trust. Name the hidden problem.
- W4: Sensory interior moment. Let visuals do the work.
- W5: Math/value email. Install the comparison reframe BEFORE launch day.
- W6: Trust/credibility close. Set up launch anticipation.

### Track B (Launch Week)
- L1: T-24h. Logistical prep (create Indiegogo account, check hitch).
- L2: Launch hour. Single CTA. Minimal explanation (they've read 6 emails).
- L3: Day+1 velocity report to non-backers. Honest count, social proof.
- L4A: Conditional — Tier 1 sold out trigger. Celebrate backers, open Tier 2.
- L4B: Fallback — Tier 1 still open. Calmer tone, extend the window.

### Track C (Mid-Campaign)
- M1: Social proof roundup (backer comments, press mentions).
- M2: FAQ dissolve. Pick the #1 blocking question (usually hitch compatibility).
- M3: Stretch goal unlock. News is the email.
- M4: Factory floor / behind-the-scenes. Rebuilds trust after campaign lull.
- M5: Re-engagement for non-backers. Honest hesitation framing, no guilt.

### Track D (Last 48h)
- F1 (T-48h): Clear deadline, clear price delta vs. post-campaign.
- F2 (T-24h): Personal/founder voice. The emotional appeal, not the urgency stick.
- F3 (T-6h): Hard close. Recap core promise in one line. Last link.

### Track E (Post-Launch)
- P1 (+3d): Backer welcome. Set production update cadence. Human voice.
- P2 (+10d): Referral program. Frame as invitation, not commission.
- P3 (+30d): Production update #1. Establish monthly cadence. Honest about delays.
