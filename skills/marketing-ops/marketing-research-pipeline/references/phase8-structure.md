# Phase 8 Directory Structure and File Naming Convention

## Output Directory
```
~/.hermes/research/output/[product-name-slug]/08-desire-test-copy/
```

## File Naming Pattern
`[desire-slug]-[awareness-level].md`

### Desire Slugs
- `time-reclamation`
- `physical-comfort`
- `relationship-harmony`
- `spontaneous-adventure`
- `freedom-from-clutter`

### Awareness Levels
- `unaware`
- `problem-aware`
- `solution-aware`
- `product-aware`
- `most-aware`

### Examples
- `time-reclamation-problem-aware.md`
- `physical-comfort-product-aware.md`
- `relationship-harmony-most-aware.md`

## Copy Requirements (Prompt #8)
- 1,200-1,400 words per copy
- Personal Discovery Story format (NOT ad copy)
- Opening hook: 20 words max, irresistible
- Story arc: vulnerability → desperate search → discovery → skepticism → hope → transformation
- Short paragraphs (1-3 sentences)
- Specific details (times, names, numbers, physical sensations)
- No forum-style quoted blocks
- No marketing speak
- End with soft urgency
- The reader should feel: "This person went through exactly what I'm going through"

## Volume Rule
ALWAYS generate 5 desires × 5 awareness levels = 25 copies total. Do not stop at the dominant awareness level.
