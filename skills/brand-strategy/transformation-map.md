## About This Skill

The Transformation Map is a visual framework that charts the buyer's journey from starting point to desired outcome. It makes abstract transformations concrete and measurable, enabling both client buy-in and progress tracking.

## Key Concepts

- **Starting Point Assessment**: Where the client currently is — messaging, positioning, authority level — in specific, measurable terms
- **Destination State**: The concrete outcome, not vague goals — 'predictable growth' is vague; '20 qualified leads per month from LinkedIn' is concrete
- **Transformation Milestones**: 3-4 stages between start and finish, each with specific metrics and timelines
- **Snap Point**: The moment buyer tolerance for current workaround collapses — optimal timing for conversion push
- **Visual Delivery**: One-page document that can be referenced in all stakeholder conversations, especially CFO-level justification

## How to Use This Skill

Use when a client is unclear about the path from their current state to their desired outcome. Build the Transformation Map collaboratively — the act of defining the map is often more valuable than the map itself. It forces clarity about what success actually looks like.
