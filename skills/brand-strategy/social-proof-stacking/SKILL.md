## About This Skill

Social Proof Stacking combines four proof types — Case Studies, Testimonials, Industry Recognition, and Milestones — to create compound credibility. Each type serves a different trust mechanism: competence, human trust, borrowed authority, and momentum. Stacking them creates an authority signal no single proof type can match.

## Key Concepts

- **Case Studies**: Competence — specific client transformation with measurable outcomes
- **Testimonials**: Human trust — quotes with specific results that feel relatable
- **Industry Recognition**: Borrowed authority — third-party validation from trusted sources
- **Milestones**: Momentum — trajectory markers showing growth and scale

## How to Use This Skill

When building social proof, don't rely on one type. Stack multiple proof categories strategically. Case studies demonstrate competence through data. Testimonials create human connection through relatable results. Industry recognition borrows credibility from trusted institutions. Milestones show trajectory and scale.

Update proof regularly as results improve. A profile from 2022 won't reflect current capabilities. The stacking effect — all four types visible together — creates compound authority stronger than any single type alone.
