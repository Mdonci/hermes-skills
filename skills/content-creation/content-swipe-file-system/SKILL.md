  - Save content you find compelling from Twitter, TikTok, LinkedIn, Facebook — anywhere you scroll
  - Organize saved content into categorized swipe files by theme or content type
  - Title each saved post so you can easily find it later
  - Review your swipe file when it's time to write new content to find patterns and structure
  - Use swipe file examples as structural templates, not for copying specific content
pitfalls:
  - Saving content without proper categorization — becomes unusable later
  - Copying content directly instead of using as structural inspiration
  - Not adding your own context when using AI tools to generate variations
  - Keeping the swipe file too large — focus on posts that genuinely resonated with you
