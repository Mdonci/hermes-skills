# Dman's Hermes Skills

Custom skills for the Solo Company OS — marketing ops, copywriting, content creation, brand strategy, and operational tooling.

## Installation

```bash
hermes skills tap add Mdonci/hermes-skills
```

Then install individual skills:

```bash
hermes skills install Mdonci/hermes-skills/writing-quality
hermes skills install Mdonci/hermes-skills/marketing-ops
hermes skills install Mdonci/hermes-skills/brand-strategy
```
